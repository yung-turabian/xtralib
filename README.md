# reislib

A static library that eases the creation of C software. Built for C, but works with C++.

# About

Provides some useful functions & data structures for C/C++.

# Features

## Data Structures

+ Hash table (hash map)
+ Trie
+ Pair
+ Multi map
+ Matrix (WIP)

## Extensions

+ String operations
+ File interactions
+ Memory management superset
+ Unix operations
